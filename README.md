Highlights:

- 🌟 Tech stack: React + Mysql + TailwindCSS + Daisy UI
- 🎃 Authentification sécurisée (JWT + Argon2 + 2FA par email)
- 👌 Global state management with Zustand
- 🐞 Error handling both on the server and on the client
- ⏳ And much more!
